Welcome to NHyperCat

A work-in progress "Hypercat" library for .NET

NHyperCat is a tiny and easy to use library for working with Hypercat. Have you ever wondered on how to generate or consume Hypercat based catalouge's. NHyperCat does the job of building Hypercat 3 compatible catalogue's. Also, it has a built in support for parsing the Hypercat catalouge's too. As a data provider, NHyperCat helps in rapidly building catalouge in no time.

More information about Hypercat can be found here:

http://wiki.1248.io/doku.php?id=hypercat
