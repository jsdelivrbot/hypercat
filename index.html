Welcome to NHyperCat

A work-in progress "Hypercat" library for .NET

NHyperCat is a tiny and easy to use library for working with Hypercats. Have you ever wondered on how to generate or consume HyperCat based catalouge's. NHyperCat does the job of building Hypercat 3 compatible catalogues. Also, it has built in support for parsing the HyperCat catalouge's too. As a data provider, NHyperCat helps in rapidly build catalouge's in no time.

More information about (and the specification for) Hypercat can be found here:

http://wiki.1248.io/doku.php?id=hypercat
